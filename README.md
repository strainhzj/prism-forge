# PrismForge

A Tauri-based desktop application built with React and TypeScript.

## Project Overview

PrismForge is a modern desktop application combining the power of Rust (Tauri) with React for a responsive user interface.

## Tech Stack

- **Frontend**: React 19 + TypeScript + Vite
- **Backend**: Rust + Tauri 2
- **Build Tool**: Vite

## Development

### Prerequisites

- Node.js (v18 or higher)
- Rust toolchain
- pnpm/npm/yarn

### Install Dependencies

```bash
npm install
```

### Development Mode

```bash
npm run tauri dev
```

### Build for Production

```bash
npm run tauri build
```
